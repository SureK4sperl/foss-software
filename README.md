## Contributions

This repository is **read-only** for the public.  
If you want to suggest new software or updates, please create a **pull request**. All additions must be approved before being merged.

**How to Suggest:**
1. Fork this repository.  
2. Add your suggestion (with links & description).  
3. Open a pull request explaining why the software should be added.  
4. Wait for review and merge by the repo owner.
